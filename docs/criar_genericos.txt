# Documentação dos Componentes Genéricos

## Objetivo
Este documento define os componentes genéricos que serão criados para substituir widgets específicos do app, 
eliminando duplicação de código, garantindo consistência visual e facilitando manutenção.

## Benefícios Esperados
- Redução de 70-80% dos widgets específicos
- Consistência visual em todo o app
- Desenvolvimento mais rápido de novas features
- Manutenção centralizada
- Criação de um design system próprio

---

## 1. AppCard
**Finalidade:** Componente genérico para exibir informações em formato de card
**Propriedades:**
- title, subtitle, content, customContent
- leading, trailing, actions
- tags, status, dateText
- backgroundColor, borderColor, borderRadius
- padding, margin, elevation
- selected, enabled
- onTap, onLongPress, onDoubleTap

**Substitui:**
- note_card.dart
- habit_card.dart
- project_item.dart
- list_item.dart
- log_list_item.dart
- today_log_item.dart
- day_summary_card.dart
- daily_activity_item.dart
